# 🎮 MICROLOL ACADEMY

**MicroLoL Academy** es una actividad de Discord diseñada para entrenar las mecánicas fundamentales de juegos competitivos (como League of Legends) a través de minijuegos enfocados en la precisión y los reflejos.

## 🚀 Minijuegos Incluidos

### 1. 🗡️ Aim Faster (Entrenamiento de Precisión)
Un desafío de puntería donde el objetivo es destruir objetivos circulares que aparecen aleatoriamente en la pantalla.
- **Mecánica:** Clic izquierdo para golpear los objetivos.
- **Desafío:** Si fallas 3 objetivos, el entrenamiento termina.
- **Progresión:** La velocidad de aparición aumenta con el tiempo y depende de la dificultad elegida.

### 2. 🛡️ Dodge Skills (Entrenamiento de Reflejos)
Un simulador de esquive de proyectiles con controles inspirados en los MOBA.
- **Mecánica:** Movimiento con **clic derecho** (LoL Style).
- **Habilidades:** Usa la tecla **D** o **F** para realizar un **Flash** y teletransportarte instantáneamente.
- **Desafío:** Esquiva proyectiles de diferentes tamaños y velocidades durante el mayor tiempo posible.

## ✨ Características Principales

- **🎮 Sistema de Dificultad:** Tres niveles (Fácil, Medio, Difícil) para cada minijuego.
- **🏆 Récords Persistentes:** Tus mejores puntuaciones se guardan localmente para que puedas ver tu progreso.
- **🖱️ Personalización de Cursor:** Cambia la apariencia de tu cursor desde el menú de ajustes (Default, Cruz, Puntero, MicroLoL).
- **🔊 Gestión de Audio:** Música ambiente y efectos de sonido con control de volumen integrado.
- **🎨 Estética eSports:** Interfaz moderna y oscura con efectos visuales tipo "glow" y grids dinámicos.

## 🛠️ Tecnologías Utilizadas

- **Core:** JavaScript (ES6+), HTML5 Canvas.
- **Framework:** Vite para el desarrollo y bundling.
- **Integración:** Discord Embedded App SDK.
- **Estilo:** CSS Moderno con variables y animaciones personalizadas.
